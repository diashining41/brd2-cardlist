# 바이스슈발츠 카드리스트 (한글)

바이스슈발츠 **전 카드풀**을 레어도별로 묶고 일본어 텍스트를 한글화하는 카드 뷰어입니다.
검색·색상/종류/레벨/수록(세트) 필터, 레어도 변형 이미지 비교(라이트박스)를 지원합니다.

👉 **뷰어 열기: https://diashining41.github.io/brd2-cardlist/**

### 수록 범위
전 **163개 작품 · 64,411장**(기본카드 41,668종)의 카드 데이터·이미지를 담고 있습니다.
한글 번역은 작품 단위로 순차 진행 중이며, 홈 화면의 작품 목록에 **작품별 번역 진도**가 표시됩니다.
번역이 안 된 작품도 **일본어 원문·이미지·필터는 그대로** 사용할 수 있습니다.
